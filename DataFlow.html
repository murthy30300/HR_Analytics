HR Analytics Dashboard - Data Flow Diagram

This diagram illustrates the flow of data through your HR Analytics Dashboard project, from data acquisition to final deployment.
Level 0 DFD (Context Diagram)
Level 0 (Context) Diagram: Shows the highest-level view of the system. HR dataset is the input, processed by the HR Analytics System, with the interactive dashboard as output accessible to users/stakeholders.
Level 1 DFD (Main Processes)
Level 1 Diagram: Breaks down the system into five main processes:
  1. Data Ingestion: Raw HR dataset is imported into the system
  2. Data Preprocessing: Cleaning, handling missing values, encoding
  3. Data Visualization: Creating interactive visualizations in Power BI
  4. Dashboard Embedding: Integrating into the front-end webpage
  5. Cloud Deployment: Hosting on Microsoft Azure for user access
The diagram shows data stores for cleaned data and the Power BI dashboard, with data flows connecting all processes from raw data to final user access.

Note: This data flow diagram aligns with Section 10 of your project document, illustrating the logical flow of data from acquisition through preprocessing, visualization, and deployment to end users.
